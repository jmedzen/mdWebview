# 《真如與阿賴耶識義》




臺灣 王穆提 略撰

# 若自性身從因能所生是則有為

《解深密經》〈勝義諦相品〉云：「...。若一切法真如勝義法無我性亦異相者，是則真如勝義法無我性亦應有因，從因所生；若從因生，應是有為；若是有為，應非勝義；若非勝義，應更尋求餘勝義諦。善現！由此真如勝義法無我性不名有因，非因所生，亦非有為，是勝義諦，得此勝義，更不尋求餘勝義諦，唯有常常時，恒恒時，如來出世，若不出世，諸法法性安立，法界安住。」此中即說真如非因所生，若是從因生，則應當是有為法，若是有為法，則應當非為勝義諦，一切有為法皆為苦諦等攝，此等即知說阿賴耶識為空性者，為錯謬義，不解唯識義之諸論師於此而生興謗。

阿賴耶識者，無著菩薩《顯揚》云：「謂先世所作增長業煩惱為緣，無始時來戲論熏習為因，所生一切種子異熟識為體。此識能執受了別色根，根所依處，及戲論熏習。於一切時，一類生滅，不可了知。又能執持了別外器世界。與不苦不樂受等相應。一向無覆無記。與轉識等，作所依因。與染淨轉識受等俱轉。能增長有染轉識等為業，及能損減清淨轉識等為業。」此中即說由於先世所作增長一切惑業苦等煩惱為緣，無始時來於戲論熏習為因，故有所生一切種子異熟識作為體，故《成唯識論》云：「執受有二，謂諸種子及有根身。諸種子者，謂諸相、名、分別習氣。有根身者，謂諸色根及根依處。此二皆是識所執受，攝為自體，同安危故。」既有執受，如何說常不變義？不苦不樂受亦是受蘊所攝故，又為轉識之所依因，此中亦通生雜染義，於此又破執意識能結生相續，如《攝論》說。

故〈勝義諦相品〉又云：「云何知有此識？如薄伽梵說：無明所覆，愛結所繫，愚夫感得有識之身。此言顯有異熟阿賴耶識。」

若自性身從因能所生是則有為、一切因緣所生法，皆是緣起義，故真如勝義法無為性非因所生、非有為法所攝，無漏有為法亦屬有為非無為故，真如遍一切一味相故，滅諦所攝故、為涅槃真義故，然勝義諦雖離名言，異生有情亦需善能分別了知觀諸法性相、由是悟入一切法真如無我性。

# 無漏淨種非阿賴耶識所攝

依《楞伽經》十卷本第八卷言：「阿梨耶識名空如來藏，具足無漏熏習法故，名不空如來藏。」彼楞伽經義說，異生有情第八本識阿賴耶識能含無漏淨種，雖非藏識所攝、然名之為藏者，為當佛因故名為如來藏，然又未於得菩提道果時，
無漏淨種本俱佛性義故名如來藏，阿梨耶識性虛妄故、亦破壞故、不實名空如來藏。然其所含無漏淨法種子為異生有情未來成佛報身因故名為如來藏，性非虛妄、究竟不盡，名為不空如來藏。

# 執藏識心體不異數論宗

此生現行圓滿果位名報化身，故斷能藏有漏藏識，有言將藏識能藏性說之為心體，非也，若依藏識能藏性譬之如心體，不異數論所宗。真見道位所知障滅大智滿故證自受用身，又賴耶藏識俱能藏、所藏、執藏義故，與雜染法互為緣故，有情末那我見緣故，此文雜染能、所藏中唯有能
藏令雜染等法不失故，能攝藏雜染法，與雜染法互相攝藏故亦為有情執藏為我，以是故說此藏識名阿賴耶。

# 因位中有漏染種為能藏義

我愛緣之為執藏義，即阿賴耶識為所藏而別顯藏義，即與染法為互攝藏等是，又此能執藏亦通無漏，含無漏淨種故、真見道位證解賴耶與圓鏡智相應故。又此為所藏不通無漏於諸佛位中非所藏故，異生我執性皆已盡、不復受熏故，此中名藏故不取能藏，能藏者，唯在因中以得其名，於因位中有漏染種為能藏義。

# 清淨法界是自性身、非自受用身

又有多義：

一者、《唯識》、《佛地》
皆作是說，清淨法界是自性身、非自受用身。依《佛地經》說清淨真如為自性身，而《攝大乘論》則為依四智心品殊勝說，轉去阿賴耶識得自性身者，通《莊嚴論》說，轉第八識得圓鏡智，故合此清淨真如、第八識二法為自性身，佛地皆為無漏法故。《佛地經》說，真如清淨法界為自性身為諸佛共有，為自性常，於十地中分皆證得，《成唯識論》說十真如是其義也。若圓鏡智是自性身，便無是義，相續常故、四智品所攝、非《解深密經》勝義諦品所攝。

又依《金光明經》說如如、如如智皆名法身，若有為德本說圓鏡智是法身者，應同彼經平等性智、妙觀察智、成所作智四實智品皆是法身，然則非也。又說若轉諸轉識得受用身，如成事智品非受用身者，應非轉識如成事智品為變化身者。

又《成唯識論》卷十云：「云何四智相應心品？一、大圓鏡智相應心品。謂此心品，離諸分別。所緣，行相，微細難知。不忘不愚一切境相。性相清淨，離諸雜染。純淨圓德現種依持。能現能生身土智影。無間無斷，窮未來際。如大圓鏡，現眾色像。」圓鏡智品為心品、有所緣、行相微細難知，如《華嚴金師子章》所言重重無盡觀。

# 藏識相續常，非本性常義

另，有義，依《成唯識論》說，清淨法界為自性身，《莊嚴論》等說，自性身本性常故，《讚佛論》說，佛自性身無生滅故，無著《金剛般若論》說，受持演說彼經功德，於佛法身為證得因，於餘二身為生起因，故彼經云：一切諸佛法，皆從此經出，一切如來從此經生，諸經論說究竟轉依以為法身，轉依即是清淨真如。

故知法身法界為性，然說轉去藏識得者，謂由轉滅第八識中二障麤重，證得真如顯法身故，智殊勝中，說法身者是彼藏識依止，彼藏識實性故，既是實性，故知一切賢聖依無為法而有差別者，非依藏識相續常，真如實非四智心品攝故。

# 藏識既有心所，則是有為

又圓鏡智所起常遍色身，為自受用身，藏識於異生位體性為不可知執受、觸、了、行相：謂「了」、「不可知」者，即於所緣行相之內，差別之義，既無別，用，故非別門。若別開者，束五受門，相應中攝，俱心所故，既有心所，則是有為。

# 藏識有所緣即是有為法

藏識又有所緣：謂「執受處」謂有根身、器世間、種子。另，《莊嚴論》中說，大圓鏡智是受用佛，《攝大乘》說，轉諸轉
識得受用故，雖轉藏識實得受用，《攝大乘論》言由說轉彼而顯法身故，既說法身無生無滅，唯證因得非色、心等，則四智心品之圓鏡智品與此相違，若圓鏡智非受用身
屬何身攝？此自受用身攝佛不共有為法實德，故四智心品實有色心，總說於佛地皆自受用。然自受用身雖是無漏有為法功德，非佛共有，是各別體，非法真性，非自性身，非是無為界功德本，故不名法身，《金光明最勝王經》言：「不實之法是從緣生，真實之法不從緣起，如來法身，體是真實，名為涅槃。」一切緣起法皆是有為法，是為三乘共許義，如來法身不從緣起，故是無為法。

# 清淨法界為真如、非圓鏡智

另依《佛地經論》說，清淨法界為自性身，四智自性相應共有，常遍色身真實功德，圓鏡智為自受用身，三大無數劫所修成故，
天親菩薩《般若論》說亦名報佛，《般若論》言：「何以故？一切聖人依真如法清淨得名，以無為法得名故，以此義故，彼聖人說彼無為法。復以何義，如彼聖人所證法，不可如是說，何況如是取。何以故？彼法遠離言語相，非可說事故，何故不但言佛，乃說一切聖人，以一切聖人依真如清淨得名故，如是具足清淨如分清淨故。」則知清淨法界為真如、非圓鏡智。

# 二乘二轉依果菩提、涅槃皆名解脫身

《解深密經》、《瑜伽》卷七十八亦說有二身：一者、法身，二者、解
脫身。曼殊室利白言：世尊，聲聞、獨覺所得轉依名法身不？善男子，不名法身，名解脫身。聲聞、獨覺以依解脫身故說彼等與諸佛平等，然由法身故說有所差別，此中說二乘二轉依果菩提、涅槃皆名解脫身，離煩惱障縛義故。

# 四智菩提、清淨涅槃合名為法身

如來所證二轉依果四智菩提、清淨涅槃則合名為法身，一切無量功德勝所依故，《成唯識論》云：安樂解脫身，大牟尼名法是也，依《述記》解：「頌意所顯如後頌說安樂．解脫身．大牟尼名法故，牟尼者寂默義，寂止默靜諸雜染故，非彼菩提是唯識性，在大牟尼名法身故。」彼菩提者，四智心品是，於異生位為唯識相，佛地通無漏有為法、亦圓成實性所攝故，滿分清淨故。

# 淨分依他亦稱圓成，為普斷染法故名之

《述記》又言：「淨分有為亦具三義：

一者、離倒，體非染故，是實義。

二者、究竟，諸有漏法加行善等，不能斷惑非究竟故，諸無漏法體是無漏，能斷諸染是究竟故，是成義。

三者、勝用周遍，謂能普斷一切染法，普緣諸境緣遍真如，故言第三勝用周遍，是圓義，由具三義與真如同，故淨依他亦得稱為圓成實性。」足見淨分依他亦稱圓成實性者，為普斷一切染法故亦名之。

# 真如與阿賴耶識錯認同一者

學人將真如與阿賴耶識錯認同一者，乃第八識阿賴耶識為異生有情流轉染法作依持用、還滅淨法無漏法種執持皆依藏識，故而將真如自性身與阿賴耶識自受用身體性相混。

# 由第八識生雜染

然由此第八識故煩惱雜染、異熟業果、生雜染、世間淨成，攝持諸有漏法故，何謂生雜染？
無著菩薩《顯揚聖教論》卷一云：「謂因煩惱及業故生。因生故苦。苦復多種。謂胎藏所迫苦，老病死苦，怨憎會苦，愛別離苦，求不得苦，與麤重行俱生長苦，數死生苦，生諸難苦，是名為生。」

# 意證涅槃但取涅槃，是所依執故

然亦依此第八本識則有涅槃滅諦證得者，謂由有此第八本識故執持一切順還滅法令修行者證得涅槃故，一切異生有情身中無漏淨法種，名為順還滅法，依《瑜伽》卷五十一解，還即是四諦之道諦，滅即四諦之滅諦，即是無漏法種順還滅現行、依能證得道證滅諦故，世出世間淨若無此本識皆不得成，或但涅槃是識執持依者，雖言執持無漏法種，而意證涅槃但取涅槃，是所依執故，三性中言依他起性為遍計所執依、亦圓成實之所依也，無漏依他起性於佛地亦圓成實性所攝也，此為合說無漏法。

# 涅槃不依阿賴耶識有故

然涅槃不依此識有故，藏識恆執無漏法種非執涅槃滅諦，涅槃不親依此識故，述記言：「疏則可依，今說親故。」唯親執無漏種，涅槃滅諦是異生所求果故。或此不依此本識有涅槃故，說所證涅槃是修行者正所求故、涅槃非藏識所生故，滅諦涅槃與道諦俱是還滅法所攝故，滅諦體為還道亦順彼滅諦斷滅染法，以能順滅故，亦說是滅品類攝，是為雙說，道與涅槃俱可言還，俱可言滅故，言俱者是還滅品攝，是彼品攝，非是彼體。

# 能所斷證雖依賴耶，意指與還滅法作依持用，無漏法種不依此識有

又涅槃言顯所證滅諦，後證得言顯能得道諦，由能斷道斷所斷惑究竟盡位證得涅槃，謂由還故斷前流轉證得涅槃，是故雙取也。能所斷證皆依此本識，是與還滅法作依持用，能持無漏法種故，然無漏法種不依此識有故。

能斷道，所斷惑。能證道，所證滅。皆依此識，故與還滅為依持用。

# 唯取諸無漏者為道諦體

此現行識自性無始恒有，與雜染、清淨二法總別為所依止，雜染法者謂苦集諦，即所能趣生及業惑，
謂苦、集二諦一切有漏法，所趣苦諦，能趣集諦，生雜染、及業果、惑是也。

然生雜染、業果、惑皆通苦、集二諦，又大乘中有苦諦非集諦者，謂諸異熟無記法等，生唯苦諦，解染污法別中雖唯言趣，亦通能趣集苦，故取集諦稱之。

然清淨法者何？清淨法者謂滅、道二諦，即所能證涅槃及涅槃道，謂滅、道二諦，即一切無漏無為、有為總攝於此。

滅諦所證，道諦能證，此中唯取諸無漏者為道諦體，不取異生於加行、資糧道中諸有漏法者，加行、資糧位不證滅諦故。

# 執真如為賴耶見分者，亦同我執，執阿賴耶為真如，無異勝性外道

又彼流轉還滅染淨法皆依此識，依七轉識等理不成故，與雜染法互相攝藏，亦為有情執藏為我，故說此識名阿賴耶識，此中但破勝性為因，唯據能藏以解阿賴耶，執真如為賴耶藏識見分者，亦同我執，末那恆緣賴耶藏識見分為我故流轉生死，若執阿賴耶為真如，無異於執勝性為因論者。然今有情執此名為真我者，以執此為虛妄我故正為論說名藏故，具此義故名阿賴耶，或故名阿賴耶者言別顯藏義。

# 阿賴耶識為所知依，所知依者即為有為法

所知依者何？

世親菩薩《攝論釋》云：「此中所知依殊勝殊勝語者：所應可知，故名可知，所謂雜染清淨諸法，卽三自性。依、是因義。此所知依，卽是殊勝，故名所知依殊勝，由此殊勝，故語殊勝，此依卽是阿賴耶識，如是持業釋。」

依
無性菩薩《攝論釋》云：「所應可知，故名所知。依、謂所依。此所依聲，簡取能依雜染清淨諸有為法，不取無為，由彼無有所依義故，所依卽是阿賴耶識，是彼因故。能引彼故，如其所應。若爾；所知卽所知依？由異熟識是所知性，故不相違。」
是故能知清淨真如非所知依，真如非有為法故，阿賴耶識為有為法故是所知依，如前所述。故《成唯識論》云：「此第八識，或名所知依。能與染淨所知諸法，為依止故。」

# 執賴耶心性即心體，體能緣生一切法，相雖轉變、體不變義，則同數論

另依三時教最末法輪教所依契經，《解深密經》判阿賴耶識（阿陀那識）為《心意識相品》，非《勝義諦相品》，若說阿賴耶識心性即是心體，體能緣生一切法者，此中依心立量，則同數論建立彼大等相雖轉變而體常一不變，即所謂「相雖轉變、體不變義」此即以有漏心相轉為無漏法，無漏心體作無漏故非前體相之體滅故言性常，然失前有漏相得無漏相故名為相轉變，若言前無漏心體滅後無漏心體生，則此亦不然，以有漏為相性是無漏者。

# 分別論者執無漏法有因生，若即說心，應同數論

故《成唯識論》卷二云：「分別論者雖作是說，心性本淨客塵煩惱所染汙故名為雜染，離煩惱時轉成無漏，故無漏法非無因生。而心性言彼說何義？若說空理，空非心因，常法定非諸法種子，以體前後無轉變故。若即說心，應同數論，相雖轉變而體常一。」

# 真如無為非心之因

然《勝鬘經》等契經說心性淨者說心空理所顯真如，真如是心真實性故，非賴耶藏識體，真如無為非心之因，亦非因種子能有果法，如大乘建立六無為法之虛空法、二乘建立三無為法等，故非有漏心性是無漏，名本性淨也。

# 心性本淨義為真如

然或依他心體非煩惱名性本淨，以煩惱客塵相應惑心，擾令斷客塵而染心得解脫者，如《瑜伽》五十四說心所有法等亦爾，心心所有法皆有漏依他，故知契經所言心性本淨義為真如也。

上述所言真如與阿賴耶識差異略論竟。
