# 🪷 mdWebview — 佛典經論閱讀器

mdWebview 是一款專為**佛典經論譯注與釋記**設計的網頁端 Obsidian 風格 Markdown 閱讀器。它提供輕量、流暢、排版精美的單頁應用（SPA）介面，讓讀者能夠專注於經典的閱讀、检索與研習。

---

## ✨ 核心特色

- 📂 **Obsidian 風格檔案瀏覽器**：自動掃描 `md/` 資料夾下的多層級 Markdown 檔案，以樹狀目錄直觀呈現，並顯示各資料夾下的經論檔案數量。
- 🔍 **高效全文搜尋**：伺服器端高效檢索所有經論內容，提供關鍵字定位與前後文上下文片段（Snippet）預覽，點擊搜尋結果即可直接跳轉至該位置。
- 📑 **自動大綱導航 (TOC)**：開啟經論檔案後，自動解析 Markdown 標題（H1~H6）並動態生成側邊欄大綱，支援點擊滾動與閱讀進度追蹤（ScrollSpy）。
- 🎨 **五種精緻閱讀主題**：
  - 🌙 **Obsidian Dark** (深色科技)
  - ☀️ **Obsidian Light** (明亮清新)
  - 🔆 **Solarized** (經典護眼)
  - 🍵 **禪風 Zen** (平和淡雅)
  - 📜 **經典金 Sutra Gold** (古籍宣紙感)
- 🔎 **浮動本頁搜尋**：支援透過快速鍵喚出頁面內搜尋框（`Ctrl + F`），具備相符項目計數、高亮與前後切換功能。
- 🅰️ **動態字型縮放**：可自由調整閱讀區域字型大小（`Ctrl + +` / `Ctrl + -`），體貼不同視力需求的讀者。
- ⚡ **無依賴輕量架構**：後端基於 Node.js 原生 API，前端為原生 JavaScript，無須繁雜的建置流程，啟動即用。

## 🚀 快速開始

### 1. 環境需求
- 系統需安裝 **Node.js** (建議 v14 或以上版本)。

### 2. 安裝與執行
1. **安裝依賴**（專案後端使用原生 API，不安裝依賴亦可直接以 `node server.js` 啟動。若要使用完整腳本，可執行安裝）：
   ```bash
   npm install
   ```

2. **啟動伺服器**：
   ```bash
   npm start
   # 或
   npm run dev
   # 或直接執行
   node server.js
   ```

3. **瀏覽閱讀**：
   開啟瀏覽器並造訪 `http://localhost:8330` 即可開始閱讀。

### 3. Docker 部署 (推薦)
本專案已完整支援 Docker 與 Docker Compose 容器化部署：

1. **準備數據目錄**：
   在專案根目錄下建立 `data` 資料夾與其子目錄 `md`：
   ```bash
   mkdir -p data/md
   ```
   請將您要閱讀的經文 Markdown 檔案放入 `data/md` 目錄中。

2. **啟動容器**：
   ```bash
   docker-compose up -d
   ```
   這會自動在背景建置映像檔並啟動 mdWebview 服務，將本機的 `./data` 目錄掛載至容器的 `/data`，以便持久化保存 `config.json` 系統設定與您的經文書庫。

3. **訪問系統**：
   造訪 `http://localhost:8330`。首次進入時，系統會引導您設定管理員的帳號與密碼。設定完成後即可立即開始閱讀與使用全文搜尋！

## ⌨️ 快捷鍵指南

| 快捷鍵 | 功能說明 |
| :--- | :--- |
| <kbd>Ctrl</kbd> + <kbd>F</kbd> 或 <kbd>Cmd</kbd> + <kbd>F</kbd> | 開啟本頁搜尋框 |
| <kbd>Esc</kbd> | 關閉本頁搜尋框 |

| <kbd>Ctrl</kbd> + <kbd>-</kbd> | 縮小閱讀區域字型 |

## 📝 經論 Markdown 撰寫規範建議

為了獲得最佳的閱讀體驗，建議您的 Markdown 檔案採用以下規範：
1. **標題階層**：使用 `#`、`##`、`###` 標示章節目錄，這將被自動解析為側邊欄的大綱。
2. **註腳支援**：經典註釋常用註腳，本閱讀器支援標準 Markdown 註腳格式，例如：
   ```markdown
   論文 餘九皆通見、修所斷。[^1]
   
   [^1]: 指其餘九支皆通於見道與修道所斷。
   ```
3. **元數據 (Frontmatter)**：您可在檔案最上方加入 YAML Frontmatter，用以指定標題、作者或版本資訊，例如：
   ```yaml
   ---
   title: 金剛般若經贊述釋記
   author: 窺基大師 撰 / 後學 敬錄
   ---
   ```
